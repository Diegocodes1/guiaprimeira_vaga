🗺️ Roadmap de 90 Dias

Este plano foi criado para quem deseja conquistar sua primeira vaga em TI. O objetivo é estudar de forma consistente, criar projetos e montar um portfólio.

📅 Dias 1 a 30 — Fundamentos

Objetivo

Construir uma base sólida.

  • ✅ Lógica de Programação
  • ✅ HTML
  • ✅ CSS
  • ✅ Git
  • ✅ GitHub

Projeto

Crie uma Landing Page pessoal e publique no GitHub.

📅 Dias 31 a 60 — Desenvolvimento Web

Objetivo

Adicionar interatividade às páginas.

  • ✅ JavaScript
  • ✅ DOM
  • ✅ Eventos
  • ✅ Arrays
  • ✅ Objetos
  • ✅ Fetch API

Projetos

  • Calculadora
  • Lista de Tarefas
  • Relógio Digital

📅 Dias 61 a 90 — Preparação para vagas

Objetivo

Construir um perfil profissional.

  • ✅ SQL Básico
  • ✅ Python ou Java
  • ✅ Currículo
  • ✅ LinkedIn
  • ✅ Portfólio
  • ✅ Entrevistas

Projetos

  • Consumir uma API pública
  • Sistema de Login
  • CRUD simples

📌 Checklist Final

  • ☑ GitHub organizado
  • ☑ LinkedIn atualizado</
